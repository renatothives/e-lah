Configurações básicas

git config --global user.name "nome"
git config --global user.email "e-mail"
Git bash >> ícone >> botão direito >> propriedades >> inicar em

Comandos básicos

Criar repositório 
git init
Situação atual 
git status
Adicionar arquivos 
git add {nome-do-arquivo.extensao}
git add {*.txt}
git add .
Confirmar as mudanças 
git commit -m "{mensagem}"
Confirmar as mudanças pulando o gitadd 
git commit -a -m "{mensagem}"
Como ignorar um arquivo Criar arquivo ".gitignore" incluir os nomes dos arquivos ou pastas por linha, exemplo: 
.gitignore
temp/

Verificar alterações

Consultar alterações que não estão na stage area 
git diff
Consultar o que está na stage area 
git diff --staged
Consultar todos os commits 
git log
git log -p
git log --pretty=oneline
Limitar log's exibidos 
git log -p -{n}
git log -p -1
Melhor opção é a interface gráfica 
git k

Desfazer commit

Editar commit mais recente 
git commit --amend -m "{mensagem}"
Remover da stage area 
git reset HEAD {nome-do-arquivo.extensao}
Reverter alteração 
git checkout -- {nome-do-arquivo-.extensao}
Deletar um arquivo 
git rm {nome-do-arquivo-.extensao}

Tag é uma etiqueta ou melhor um ponto de atalho (ponteiro)

Listar as tag's 
git tag
Criar uma tag anotada (-a) 
git tag -a {nome-tag} -m {mensagem}
Criar uma tag num determinado commit 
git tag -a {nome-tag} {chave-completa} -m {mensagem}
Detalhes de uma determianda tag 
git show {nome-tag}
Usar arquivos como eram na criação da tag 
git checkout {nome-tag}
Voltar para versão atual dos arquivos 
git checkout master
Deletar uma tag 
git -d {nome-tag}

Branch's são ramificações

Criar um branch 
git branch {nome-branch}
Transitar para um branch 
git checkout {nome-branch}
Criar um branch e já transitar os arquivos para ele 
git checkout -b {nome-branch}
Unificar alterações de um branch para o atual 
git merge {nome-branch-origem}
Deletar um branch 
git branch -d {nome-branch}
Listar todos os branch's 
git branch

Trabalhar com ambiente colaborativo

Copiar um repositório para a pasta local 
git clone {caminho} {nome-pasta-personalizada}
